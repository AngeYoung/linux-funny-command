# linux-funny-command
40个超有趣的Linux命令行彩蛋和游戏，一键安装脚本    
按顺序执行以下五条命令即可
```
sudo apt-get update
git clone https://github.com/TommyZihao/linux-funny-command.git
cd linux-funny-command
sudo chmod 777 1.sh
./1.sh
```
